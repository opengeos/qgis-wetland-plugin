about=Wetland Mapper is a native QGIS plugin for wetland mapping and analysis.

    It ships with a Playa Wetlands example preset, but the plugin is designed
    for broader wetland mapping workflows. Users can load built-in data
    catalogs, add custom WMS/XYZ/PMTiles/local vector sources, browse NAIP
    imagery, inspect wetland-oriented attributes, and run JRC water occurrence
    statistics for map extents or selected features.

    Requirements:
    - QGIS/GDAL with PMTiles vector driver support for PMTiles catalog layers
    - matplotlib for chart PNG exports (installable from Settings)
